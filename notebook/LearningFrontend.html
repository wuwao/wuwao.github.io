1. element

block elements:
 a new block-level element appear on new line.

    for example: 'p'.

one paragraph

the other paragraph

inline elements:
 new inline element after last element, on same line.

    example: 'a', 'em', 'strong'.

one paragraph one strong word still same paragraph.

empty / void element
 no closing tag

    for example: 'img'.

<img src="https://...">

2. attribute

 extra information about element

    for example: 'class'.

<p class="anyclass"> a paragraph with one attribute</p>

    for example: 'a' is hyperlink, it has three attributes.(href/title/target)

boolean attribute

    for example: 'disable'.

entire HTML page

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>My test page</title>
</head>
<body>
<p>This is my page</p>
</body>
</html>

    '!DOCTYPE html' is a standard, just write it down, don't ask why.
    'html' wrap the all page, just write it down.
    'head' is the content don't display on the page.
    'meta charset="UTF-8"' include most characters from the vast majority of human written languages
    'title' is the title of the page.
    'body' is the content display on the page.
